## Installation

```bash
$ yarn install
```

# Cambiar las variables de entorno

```
docker-compose up -d
```

# Ejecutar SEED

```
http://localhost:8080/api/seed
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
